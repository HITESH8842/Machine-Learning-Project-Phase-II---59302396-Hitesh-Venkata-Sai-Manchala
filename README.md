# Fashion Attribute Recognition Using CNNs

## Project Overview

This project develops a Fashion Attribute Recognition system using Convolutional Neural Networks (CNNs) and Transfer Learning techniques. The objective is to automatically classify fashion product images into different article categories using deep learning models. The project is part of the Machine Learning course and focuses on applications in Retail and E-Commerce Visual Intelligence.

## Project Objectives

* Classify fashion product images using CNNs.
* Apply image preprocessing and data augmentation techniques.
* Compare a custom CNN model with a MobileNetV2 transfer learning model.
* Evaluate model performance using multiple evaluation metrics.
* Generate visualizations and performance reports.

## Dataset

**Dataset Name:** Fashion Product Images Dataset

**Source:** Kaggle

**Dataset Link:** https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-dataset
